Project scope

Plus One is a Django-native MVP for spontaneous campus matching.

It demonstrates multiple pages, user-created database records, swipe interactions, countdown timers, anonymous chat, and AI-assisted posting without npm or extra servers.

Database interaction

Users create posts, swipe, generate matches, send messages, and write AI logs. Seed data only makes the live demo easier to start.

AI integration

OpenAI is used when OPENAI_API_KEY is available. Otherwise the app uses deterministic fallback logic so the demo still works offline.

MVP path

Create → AI card → Publish → Switch user → Swipe → Match → Chat → Agree → Dashboard.
